# Large-Scale Network Optimization

Analysis of structural robustness in the U.S. domestic airline network using max-flow and min-cut methods. Built for IEOR 240 at UC Berkeley.

## What It Does

Models the U.S. domestic airline network as a directed, capacitated graph and stress-tests its resilience under capacity constraints and hub disruption scenarios.

- Graph construction from FAA flight data using NetworkX
- Max-flow / min-cut analysis to identify critical bottlenecks
- Hub disruption simulations to measure network fragility
- Seat-capacity modeling for realistic throughput constraints

## Stack

- **Language:** Python
- **Graph library:** NetworkX
- **Data:** FAA domestic flight routes and seat capacity
